# Tower Defense Simulator Macro by Darksen   
Started on March 30, 2026. My friend bet me that I wouldn't make a macro for TDS, but I did. Now he doesn't play TDS 😂

# Features:
 - Record & Play strategies - Record your tower placements, upgrades, and actions, then replay them automatically
 - Auto-Restart System - Automatically detects game end and rejoins with intelligent restart handling
 - Multiple Upgrade Paths Support
 - Discord Webhook support
 - Reconnecting upon disconnect
 - And much more!

# Requirements

- **Windows 10/11**
- **Screen Resolution:** 1920x1080 (strictly required)
- **Windows Scaling:** 100% (required)
- **Taskbar:** Must be visible (not auto-hidden)
- **Roblox & TDS Settings:**
  - UI Scale: Large
  - Screen Shake: Disabled
  - Prefer Vertical Upgrades: Disabled
  - No custom fonts
  - Chat must be closed

# Installation

1. Download the latest release
2. Extract all files to a folder
3. Run `Main.ahk` (requires [AutoHotkey v1.1+](https://www.autohotkey.com/))
